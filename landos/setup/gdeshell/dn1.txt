
    gdeshell eh um interpretador de comandos que pertence
    ao ambiente de setup.
    Ele usa a rtl e a libcore.
    Serve para executar alguns programas e realizar alguns testes de sistema
    usando comandos embutidos.

    gdeshell é uma aplicação do sistema. 
    Eh um prompt de comandos no estilo unix.
    Eh uma aplicação em user mode 32bit.

    gramci/shell/cmd
    A pasta cmd/ contém os programas principais que são chamados pelo 
    gdeshell quando executa comandos comuns.
    Programas bem simples podem ser criado e ficarão nessa pasta.
